------------------------             Manual rapido             ------------------------
- Use as teclas 'w', 'x', 'a', e 'd' para mover o jogador.
- Use 'q' pasar sair



------------------------             Caracteristicas             ------------------------
- Funcao que salva log de acoes
- Funcao que salva log de erros (duas versoes; uma com struct)
- Salva e retorna dados do jogador (nao finalizado)
- Salva dados de partidas anteriores (nao finalizado)
